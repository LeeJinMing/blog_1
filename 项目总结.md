```
├── package.json                # 项目配置
├── tailwind.config.js         # Tailwind配置
├── tsconfig.json              # TypeScript配置
├── next.config.js             # Next.js配置
├── postcss.config.js          # PostCSS配置
├── env.example                # 环境变量示例

└── README.md                  # 项目文档
```

## 🎨 设计系统

### 配色方案
- **主色调**: 蓝色系 (#3b82f6)
- **成功色**: 绿色系 (#10b981)
- **警告色**: 橙色系 (#f59e0b)
- **中性色**: 灰色系 (多层级)

### 字体系统
- **主字体**: Inter - 清晰易读的无衬线字体
- **标题字体**: Lexend - 现代化的显示字体
- **代码字体**: 系统等宽字体

### 组件规范
- **圆角**: 8px, 12px, 16px (渐进式)
- **阴影**: 多层级阴影系统
- **间距**: 4px 基础单位，Tailwind 标准
- **动画**: 300ms 标准过渡时间
